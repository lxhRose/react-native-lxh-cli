react-native-lxh-cli
===
简介
===
这是一个基于 react-native-cli 的react-native项目构建脚手架；同时带有模板文件。  

安装：  
---
    npm install -g react-native-lxh-cli
    
如果你还没有安装过yarn和react-native-cli，请先安装：

    npm install -g yarn react-native-cli   
使用
---
    newrn init projectName

本地测试
---
    npm link  
    newrn init projectName
